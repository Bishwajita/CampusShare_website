CampusShare- A resource sharing platform

CampusShare is a web-based platform designed for students to share resources within their campus community.

* Features
- Upload products
- User registration and login system
- Organized resource sharing

* Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

* Project Structure
- /templates → HTML files
- /css → Stylesheets
- Backend logic using PHP
- MySQL database for storing user data
